⏺ keybinds.luaに設定されている全キーバインドをまとめます：

  リーダーキー

  Ctrl + q（2秒間有効）

  ---
  Workspace関連

  | キー               | 動作                          |
  |--------------------|-------------------------------|
  | Leader + w         | Workspace一覧を表示・切り替え |
  | Leader + $         | Workspaceの名前変更           |
  | Leader + Shift + W | 新規Workspace作成             |

  タブ操作

  | キー       | 動作           |
  |------------|----------------|
  | Cmd + t    | 新規タブ作成   |
  | Cmd + w    | タブを閉じる   |
  | Leader + { | タブを左に移動 |
  | Leader + } | タブを右に移動 |

  Pane操作

  | キー             | 動作                                               |
  |------------------|----------------------------------------------------|
  | Leader + d       | 縦分割                                             |
  | Leader + r       | 横分割                                             |
  | Leader + x       | Paneを閉じる                                       |
  | Leader + h/j/k/l | Pane間移動（左/下/上/右）                          |
  | Leader + z       | 選択中Paneをズーム                                 |
  | Ctrl + Shift + [ | Pane選択モード                                     |
  | Leader + s       | Paneサイズ調整モード（h/j/k/lで調整、Enterで終了） |
  | Leader + a       | Pane移動モード（1秒間有効）                        |

  コピー・ペースト

  | キー       | 動作                                |
  |------------|-------------------------------------|
  | Cmd + c    | コピー                              |
  | Cmd + v    | ペースト                            |
  | Leader + [ | コピーモード開始（Vimライクな操作） |

  フォントサイズ

  | キー     | 動作     |
  |----------|----------|
  | Ctrl + + | 拡大     |
  | Ctrl + - | 縮小     |
  | Ctrl + 0 | リセット |

  その他

  | キー                            | 動作                   |
  |---------------------------------|------------------------|
  | Cmd + p または Ctrl + Shift + p | コマンドパレット       |
  | Alt + Enter                     | フルスクリーン切り替え |
  | Ctrl + Shift + r                | 設定再読み込み         |

  ---
  コピーモード内の操作（Leader + [で開始）

  Vimライクな操作が可能：
  - h/j/k/l - カーソル移動
  - w/b/e - 単語単位移動
  - 0/^/$ - 行頭/行末移動
  - g/G - 最上部/最下部
  - Ctrl+f/b - ページ移動
  - v/V/Ctrl+v - 選択モード（文字/行/ブロック）
  - y - コピー
  - q/Escape - コピーモード終了
